# Website to PDF Archive

هر وبسایتی رو با یک کلیک به PDF تبدیل کن، با حفظ کامل استایل و تصاویر.

## چطور استفاده کنم؟

۱. فایل `website-to-pdf.yml` رو بذار تو مسیر `.github/workflows/` ریپوت
۲. برو به تب Actions
۳. Website to PDF Archive رو انتخاب کن
۴. Run workflow رو بزن
۵. آدرس سایت، عمق و اسم فایل رو وارد کن
۶. منتظر بمون تا تموم بشه
۷. PDF تو ریشه ریپوت ذخیره میشه

## تنظیمات

- **target_url**: آدرس سایتی که میخوای PDF بشه
- **depth**: چند لایه لینک رو دنبال کنه (۱ تا ۳)
- **filename**: اسم فایل PDF خروجی
